I need to talk to someone

Please contact the following people depending on your needs:

Type of issue Who How Details
Service unit allocation for running jobs Sydney Scheme managers Make a request Sydney Documentation
Request for increased g/data storage Sydney Scheme managers Send an email Sydney Documentation
Request for increased scratch space NCI Helpdesk Log an NCI ticket Provide brief justification for the need including number of files, size of files, data lifecycle
Request for increased iNode quota in gdata or scratch NCI Helpdesk Log an NCI ticket Provide brief justification for the need including number of files, size of files, data lifecycle, efforts to reduce iNode footprint
A technical issue with NCI systems or compute jobs NCI Helpdesk Log an NCI ticket Provide all relevant information such as your error, log file, jobid
A bug in an SIH pipeline SIH Submit an issue on Github Provide details e.g. this issue
General research computing or data analysis advice SIH Log an SIH ticket Provide relevant context, errors, tool names, scripts
